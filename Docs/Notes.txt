[To enable CORS for all origins]

how to add CORS settings to asp.net core project - https://stackoverflow.com/questions/31942037/how-to-enable-cors-in-asp-net-core

or, follow the steps shown ahead:
 - Install the Microsoft.AspNetCore.Cors NuGet package
 - Inside Startup ConfigureServices, add the following code to implement a policy for CORS that would allow all origins:
	services.AddCors(options =>
	{
	  options.AddPolicy("AllowAll",
	    builder =>
	    {
	      builder
	      .AllowAnyOrigin()
	      .AllowAnyMethod()
	      .AllowAnyHeader();
	    });
	});

[Adding basic authentication]

2. add MyAuthnticationHandler : AuthenticationHandler<MyOptionClass>

override 2 method - 
protected override Task<AuthenticateResult> HandleAuthenticateAsync()
protected override async Task HandleChallengeAsync(AuthenticationProperties properties) - 

If a client tries to access a restricted resource or a resource that requires authentication, it's the server's responsibility to inform the client about the authentication type and related information. The WWW-Authenticate HTTP response header is set by the server that defines the authentication method that should be used to gain access to the restricted resource requested.
